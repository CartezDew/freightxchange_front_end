# FreightXchange


**FreightXchange** is a full-stack web platform designed to connect brokers and carriers in the freight industry. It streamlines how loads are posted, bids are submitted, and deals are made. With built-in bid management and profile-based role views, FreightXchange removes the friction from freight booking and helps logistics professionals get work done—fast.

---

## ✅ MVP Features

**🔐 Role-Based Authentication**  
- Secure login and registration for brokers and carriers  
- Profile-specific dashboards based on user type  

**📦 Load Posting (Broker View)**  
- Brokers can create, edit, and delete loads  
- Key fields include pickup/delivery cities, equipment type, commodity, and rate  

**💸 Bid Submissions (Carrier View)**  
- Carriers can view available loads and submit bids  
- Edit or delete their own bids with real-time feedback

**🧾 Bid Management (Broker View)**  
- Brokers can review, accept, or reject carrier bids  
- Full visibility into all submitted offers on each load  

**📱 Responsive Interface**  
- Modern and mobile-friendly UI built with Material UI and React Router  

---

## 🧩 System Architecture

**Frontend:** React + React Router + Material UI  
**Backend:** Django + Django REST Framework  
**Database:** PostgreSQL  
**Auth:** Token-based authentication (JWT)

## 🚀 Future Enhancements

1. Map Integration: Display load origin/destination on a map using Google Maps API.

2. Smart Matching: Suggest loads to carriers based on location and truck type.

3. Ratings and Reviews: Allow brokers and carriers to rate each other after job completion. 

4. Notifications: Create a preferred lanes profile for carriers. Notify carriers when new loads match their preference. 

5. AI Load Matching: Automatically suggest loads to carriers based on their historical routes and preferences.
